## How to contribute ?

### Github (Recommended)

- Fork repository
- Edit files, you can use [translatool](https://github.com/Dysnomia-studio/translatool) to help you.
- Create a pull request

### Discord

Join our [Official Discord](https://discord.gg/c8aARey), and send your translations in the **Logistik** channel.

### Steam Forum

Participate on the official [Translation Thread on Steam](https://steamcommunity.com/app/3928180/discussions/0/680733951028046258)
